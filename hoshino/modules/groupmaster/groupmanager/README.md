# 安装

0. 克隆本项目 ``` cd ~/hoshino/modules/groupmaster/ && git clone https://github.com/pcrbot/group-manager.git```
1. 重命名文件夹 ```rename group-manager groupmanager```
2. 设置龙王为自己时所发送的图片 ```mv ./dk_is_me.jpg ~/res/img/```
3. 重启HoshinoBot
